Program korzysta z moich autorskich algorytmów sortowania, stworzyłem je po zapoznaniu się z ideą działania algorytmu. Sortowanie bąbelkowe jest beznadziejne - potrzebuje około 35k - 40k operacji to posortowania tablicy, kiedy kubełkowe tylko 400, Sortowanie bąbelkowe minimalnie wykonuje wielkoscTablicy * 2 - 1 operacji, kiedy tablica jest posortowana. Sortowanie kubełkowe wykonuje zawsze wielkoscTablicy * 2 operacji przy sortowaniu. Do tworzenia losowej tablicy użyłem bilbioteki "java.util.Random". UI jest przejrzyste i wolne od błędów, zastosowałem różne kolory tekstu, żeby łatwo było odróżnić ważne informacje od menu. Każda ważna informacja jest oddzielona "\n" (next line) dla większej czytelności wyświetlanego tekstu. W środowisku Intellij nie da sie czyścić konsoli - szkoda. Interfejs przechowuje deklaracje 4 metod: int[] sort(int[] tab); - metoda sortująca void stepCounting(); - metoda licząca ilość operacji sortowania "count" void printCount(); - metoda drukująca komunikat o ilości operacji void printSorted(int[] tab); - metoda drukująca zawartość tablicy po 50 elementów na wiersz Program jest zbudowany w ten sposób, że dowolnie możemy zmieniać wielkość tablicy i dalej będzie funkcjonował poprawnie.
